🐍 TextBlob
Rule-based

Lightweight, no training required. Great for quick prototyping.

# pip install textblob from textblob import TextBlob text = "The mobile features are Amazing" blob = TextBlob(text) print(blob.sentiment) # Sentiment(polarity=0.6, subjectivity=0.8) print(blob.sentiment.polarity) # 0.6 print(blob.sentiment.subjectivity) # 0.8
Score interpretation
Polarity−1 (very negative) → +1 (very positive)
−10 neutral+1
Subjectivity0 (objective) → 1 (subjective opinion)
0 objective1 subjective
Batch analysis
reviews = [ "The product is fantastic!", "Worst purchase ever.", "It's okay, nothing special." ] for r in reviews: blob = TextBlob(r) pol = blob.sentiment.polarity label = "Positive" if pol > 0.05 else "Negative" if pol < -0.05 else "Neutral" print(f"{label} ({pol:.2f}): {r}")
💡 TextBlob uses a fixed lexicon — it doesn't understand context, so "not bad" may incorrectly score as negative. For nuanced text, prefer HuggingFace transformers.
🤗 HuggingFace Transformers
BERT-based

State-of-the-art results with 2 lines of code. Powered by pre-trained BERT / DistilBERT.

Single review
# pip install transformers torch from transformers import pipeline classifier = pipeline("sentiment-analysis") result = classifier("The mobile features are Amazing") print(result) # [{'label': 'POSITIVE', 'score': 0.9998}]
Batch reviews with zip
reviews = [ "The movie is average", "The story is bad", "Songs are Excellent", "Screenplay is worst" ] results = classifier(reviews) for text, sentiment in zip(reviews, results): print(f"Review: {text}") print(f"Sentiment: {sentiment['label']} ({sentiment['score']:.4f})")
Expected output

"The story is bad" NEGATIVE 0.99
