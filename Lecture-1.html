Part 01
Evolution of AI

AI has evolved through distinct phases — from rigid rule-based systems to modern models that generate new content on demand.

Early Days
Rule-Based & Expert Systems
Followed predefined hardcoded rules — expert systems, symbolic programming, decision trees. Example: early medical diagnosis programs. Could not learn from data. Failed in complex, real-world environments.
1990s
Machine Learning Revolution
Shift from hardcoded rules to learning from data. Key developments: statistical models, neural networks, data mining. Systems could now improve by observing patterns — without explicit instructions for every case.
2010s
Deep Learning Era (Big Data + GPUs)
Deep learning existed in theory from the 1990s but required huge computational resources. The arrival of Big Data, Hadoop, Spark, and GPUs made it viable. Breakthroughs: image recognition, speech recognition, language translation, self-driving technology.
2020s
Generative AI & Large Language Models
Modern AI systems can now generate text, create images, write code, and compose music. Powered by Transformer architectures and LLMs (ChatGPT, Gemini, Claude). A fundamental shift from "classify existing data" to "create new data".
Now & Future
Agentic AI & Autonomous Agents
AI evolving from tools that follow instructions into systems that plan, decide, and act autonomously. Future: understanding human emotions, assisting scientific discoveries, improving healthcare and education.

🤖 Definition of AI (from the lecture):

Developing systems that can perform tasks which require human intelligence — like learning, problem solving, and decision making.

Part 02
What Is Generative AI?

A subset of AI designed to create new, original content — resembling human-made work — by learning patterns from massive datasets.

🎨 Generative AI

Creates New Content

Instead of just analyzing or classifying existing data, generative AI produces entirely new outputs based on what it learned from training data.

  • Trained on large datasets (text, images, audio)
  • Learns patterns, relationships, structures
  • Generates new content from user prompts
  • Built on Transformer architectures
  • Output resembles human-made work
🔍 Traditional AI

Analyzes & Classifies

Traditional AI focuses on analyzing and predicting or classifying data that already exists — outputting a label, decision, or alert.

  • Spam detection → Spam / Not Spam
  • Fraud detection → Fraud / Legitimate
  • Image recognition → What is in this image?
  • Sentiment analysis → Positive / Negative
  • Output: a decision, label, or score
Task Traditional AI Generative AI
Email Spam detection (classify) Write the email (create)
Images Image recognition (label) Image creation (generate)
Text Sentiment analysis (score) Article / report writing
Fraud Fraud detection (flag) Report generation (summarise)
Output type Label / prediction / decision Text / image / audio / code / video
Part 03
Real-World Use Cases

Generative AI is already transforming industries — from content creation to healthcare, education to software development.

✉️
Content Writing
Emails, blogs, reports, summaries. Example prompt: "Write an email requesting leave for 5 days" → AI generates a professional, polished email instantly.
💻
Code Generation
Write Python, Java, SQL code for any task. Example: "Generate Python code for a CNN model" → AI writes the full runnable code. Deep learning models (ANN, CNN, RNN) included.
🤖
Chatbots & Virtual Assistants
Answers questions, explains concepts, assists users. Example: "Explain linear regression" → detailed explanation. Includes tools like ChatGPT, Gemini.
🖼️
Image Generation
Create realistic or artistic images from text descriptions. Different specialised models available: DALL·E, Midjourney, Stable Diffusion, Adobe Firefly.
🎵
Audio & Music Generation
Create songs, sound effects, voice-overs, and AI-generated background music for videos. Tools: MusicLM, ElevenLabs. Different models for different media types.
🏥
Healthcare
Patient in ICU for 30 days → new doctor can't read all records → AI creates a draft patient summary covering condition, treatment, improvements. Assists report generation.
🎓
Education
Generate notes, quiz questions, personalized learning material, and even PPTs. Example: "Generate practice questions for NLP topics" — instantly covers breadth.
📣
Marketing & Ads
Create ad copy, design promotional images (bank schemes, insurance plans), generate social media posts. Automates the creative cycle for any campaign.
🎬
Video Generation
Full video creation from text prompts. Tools: OpenAI Sora, Runway. Short films, ad videos, educational content. Separate models specialised for video.

❓ Q: Can one model do audio + video + text?

Different specialised models are available for different content types: video generation (Runway, Sora) · audio/music (MusicLM, ElevenLabs) · code (GitHub Copilot, Ghostwriter) · images (DALL·E, Midjourney). Multimodal models like GPT-4o and Gemini handle multiple modalities in one model.

Part 04
Generative AI vs Agentic AI

Agentic AI is the next advancement — it goes beyond creating content to planning, deciding, and executing multi-step tasks to achieve a goal.

👨‍🍳
Hotel Chef
Cooks based on your exact instruction. "Make it spicy" — he follows. Without your instruction, he doesn't act. One task at a time, on demand.
= Generative AI
🧑‍💼
Restaurant Manager
"Lunch for delegates tomorrow." — Manager prepares the menu, sources ingredients, assigns staff, ensures everything is served. A series of decisions and actions independently.
= Agentic AI
✈️
EXAMPLE: VACATION PLANNING AGENT

You say: "Plan a vacation to Goa" — the Agentic AI:

1
Searches for flights and compares prices
2
Compares hotel prices near tourist spots
3
Creates an itinerary with tourist attractions
4
Books reservations once you approve — all autonomously
🛎️
Customer Support Agent
Receives complaint → checks order status → processes refund → updates record. Companies are using AI agents to handle the entire customer support lifecycle automatically.
💻
Software Dev Agent
Reads requirements → generates code → runs tests → fixes bugs → deploys the application. Example: "Build a login page" — fully automated development loop.
📊
Data Analysis Agent
"Why did sales drop last month?" → collects sales data → performs analysis → creates charts → identifies causes → generates report → recommends next steps. All automated.

🎯 Key formula: Agentic AI = Generative AI knowledge + Reasoning + Planning + Action execution. It creates, decides, and acts to achieve a goal.

COMMON CONCERN: "Will AI take my job?"

Short answer: No — if you upskill.

AI is helping people do their jobs faster — not replacing them. A task that takes 10 hours can be done in 3–5 hours with AI assistance. People with strong AI skills are the most in-demand, highest-salaried professionals. Today there are very few people with a thorough command of these technologies — the ones who build those skills now will have a significant competitive edge.

Available roles: AI Engineer · GenAI Engineer · Data Scientist · ML Engineer · NLP Engineer · Data Analyst

🐍
WHY YOU STILL NEED TO LEARN PYTHON

"AI generates code — why learn Python?"

Python is the mother tongue of AI. If AI generates Tamil code and you don't know Tamil, you can't read it. If AI generates Python code and you don't know Python, you can't understand, debug, or validate it. You need to know: what is a function? a module? a class? a list? AI makes work faster — it doesn't replace the need for foundational knowledge.
